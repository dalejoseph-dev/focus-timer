## Mail Merge Setup (Gmail)

If sending to many candidates, use a mail merge tool:

### Option 1: Gmail + Sheets (Free)

1. Install "Yet Another Mail Merge" (YAMM) Chrome extension
2. Create Google Sheet with: Name, Email, Link
3. Draft email in Gmail
4. Use YAMM to personalize and send

### Option 2: Mailchimp (Free tier)

1. Import candidate list
2. Create email template
3. Use merge tags: `*|NAME|*`, `*|LINK|*`
4. Send campaign

### Option 3: Manual (Small batch)

If <20 candidates:
- Just send individually
- Copy/paste template
- Personalize name and link
- Takes 5-10 minutes total

---

## Timing Best Practices

**Email 1 (Invitation):**
- Send Monday-Wednesday morning
- Gives them full week + weekend
- Avoid sending Friday (weekend starts)

**Email 2 (Reminder):**
- Auto-sent on Day 5
- Or manually send if workflow fails

**Email 3 (Interview):**
- Send within 24 hours of deadline
- Don't make them wait - they're excited!

**Email 4 (Rejection):**
- Send within 48 hours of deadline
- Be kind but clear

**Email 5 (Trial Start):**
- Send immediately after hiring decision
- Give 2-3 days notice before start date
